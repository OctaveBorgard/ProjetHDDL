This project was developed by Quoc-Bao Do, Thi-Bao-Ngoc Nguyen, Mata-Dramane Sidibe and Octave Borgard.

## Projects Overview

### CVAE
Example and detailed explanation of a **Conditional Variational Autoencoder (CVAE)**, including its architecture and how it learns latent representations.

### Dog vs Cat
Example of an image processing project covering both **classification** and **segmentation**, with explanations of the methods used.

### ViT vs CNN
Comparative illustration of how **Convolutional Neural Networks (CNNs)** and **Vision Transformers (ViTs)** work, highlighting their key differences in architecture and behavior.
